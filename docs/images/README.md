# 스크린샷 폴더

이 폴더에는 다음 용도의 이미지를 넣을 수 있습니다.

- **GUI 실행 화면**: 프로그램 메인 창 스크린샷 3~5장
- **설치 가이드용**: `INSTALL_GUIDE.md`에서 참조하는 이미지
  - `macos-warning.png`, `macos-right-click-open.png`
  - `windows-smartscreen.png`, `windows-smartscreen-detail.png`

스크린샷을 추가한 뒤 랜딩페이지(`docs/index.html`)의 비포/애프터 섹션에서 경로를 지정해 사용할 수 있습니다.
